# Lucky Test Analysetool

Lokales Analyse-Dashboard für die iOS-App **Lucky Test**. Das Tool soll App-Store- sowie Social-Media-Kennzahlen aus TikTok, Instagram und YouTube übersichtlich zusammenführen.

## Aktueller Stand

Das Projekt befindet sich in der Anforderungs- und Planungsphase. Die fachliche Abgrenzung, Datenstrategie und der MVP-Umsetzungsplan stehen in [ANFORDERUNGSANALYSE.md](ANFORDERUNGSANALYSE.md).

## Vorgesehener MVP

- lokale Desktop-Anwendung für Windows
- CSV-Import für App Store, TikTok, Instagram und YouTube
- Speicherung in einer lokalen SQLite-Datenbank
- Dashboard mit aktuellen Kennzahlen, zeitlicher Entwicklung und Kanalvergleich
- keine Cloud-Kosten und keine Zugangsdaten in der Anwendung nötig

## Technische Basis

- Python 3.11 oder neuer
- Tkinter für die Oberfläche
- SQLite und CSV aus der Python-Standardbibliothek
- keine externen Python-Abhängigkeiten im ersten MVP
